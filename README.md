# JustForFun

A fun programming repository written in Go! 🚀

## About

This is a collection of fun programming exercises, experiments, and utilities written in Go.

## Getting Started

### Prerequisites

- Go 1.24+ installed on your system

### Running the Program

```bash
go run main.go
```

### Building the Program

```bash
go build -o justforfun main.go
./justforfun
```
